Ejemplo SyncAdapter

Este tutorial y aplicación se desarrollan en contexto de investigación para MECOLAB UC. Se implementa una simple aplicación de lista de alumnos de un curso. Esta se sincroniza con un servidor utilizando un SyncAdapter.

Tutorial SyncAdapter

El tutorial se puede encontrar en este link. Los temas que abarca el tutorial son:

  1. Base de datos
  2. Content provider
  3. Authenticator
  4. Crear SyncAdapter
  5. Correr SyncAdapter
